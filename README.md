Sante Price Index Project

The Sante Price Index Project is an Android-based application designed to track, analyze, and display price index data for essential goods and services. The app helps users understand price changes over time and supports better financial awareness and decision-making.Sante Price Index is an Android application designed to help users track and compare daily essential product prices in their local area. The app provides a simple and user-friendly interface to monitor price changes and understand market trends.

 Features
- Real-time price index tracking
- Category-wise price comparison 
- Simple and interactive UI
- Alerts for major price changes 
- Android-friendly responsive design

Tech Stack
- Language:Java / Kotlin 
- IDE:Android Studio
- Database:Firebase / SQLite 
- UI: XML Layouts
- Build Tool: Gradle
